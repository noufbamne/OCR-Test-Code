# 🔍 OCR Text Extractor
> Python-based tool for extracting text from scanned PDF files.

---

## 🚀 Overview
A simple OCR-based script designed to extract text from scanned documents, with a focus on use cases like soil reports and similar data-heavy files.

- 📄 Extract text from scanned PDFs  
- ⚡ Lightweight and easy to use  
- 🧪 Tested on real-world documents  

---

## 🛠 Tech Stack
- Python  
- OCR libraries (e.g., Tesseract)  

---

```bash
python code.py
```

---

## 🎯 Use Case
Useful for digitizing and analyzing data from scanned reports such as agricultural or soil documents.
